# 🐾 Animal Classifier

A deep learning model that identifies animals from photos with **94.6% accuracy**.

## Animals it can detect
🐕 Dog | 🐴 Horse | 🐘 Elephant | 🦋 Butterfly | 🐔 Chicken
🐈 Cat | 🐄 Cow | 🐑 Sheep | 🕷️ Spider | 🐿️ Squirrel

## Tech Stack
- Python
- TensorFlow / Keras
- MobileNetV2 (Transfer Learning)
- Gradio (Web Interface)
- Trained on 26,000 images

## How to run locally
pip install tensorflow gradio pillow numpy
python app.py

## Model
Trained for 10 epochs on the Animals-10 dataset from Kaggle.
